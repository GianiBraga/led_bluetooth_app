Guia Interativo: Arduino e Bluetooth

Controle de LED com Módulo Bluetooth HC-05

Um guia passo a passo para seu primeiro projeto de automação sem fio.

Parte 1: O que é o Módulo HC-05?

O HC-05 é um módulo de comunicação Bluetooth que funciona como uma "ponte" sem fio entre seu projeto Arduino e outro dispositivo, como um smartphone. Ele se comunica com o Arduino via comunicação serial, tornando simples o envio e recebimento de comandos.

Principais Pinos:

  • VCC: Alimentação (conectar no 5V do Arduino).
  • GND: Terra (conectar no GND do Arduino).
  • TXD (Transmissor): Envia dados para o Arduino (conectar no pino RX do Arduino).
  • RXD (Receptor): Recebe dados do Arduino (conectar no pino TX do Arduino).

Parte 3: O Código para o Arduino

Este código utiliza a biblioteca SoftwareSerial para criar uma porta de comunicação virtual, evitando conflitos com a porta USB. Carregue-o na sua placa Arduino.



int portas_led[2] = {8, 7 };
int led = 8;

void setup() {
  // Definindo os pinos
  for (int pinos_led = 0; pinos_led < 2; pinos_led++) {
    pinMode(portas_led[pinos_led], OUTPUT);
  }

  Serial.begin(9600);
}

void loop() {
  // Receber os dados
  char c = Serial.read();
  
   digitalWrite(led, HIGH);

  if (c == 'A') {  // Quadrado
    digitalWrite(portas_led[1], HIGH);
  }

  if (c == 'a') {  // Triangulo
    digitalWrite(portas_led[1], LOW);
  }

Parte 4: Configuração e Teste

  1. Ligue o Circuito: Após carregar o código, alimente o Arduino. O LED do HC-05 piscará rapidamente.
  2. Pareamento Bluetooth: No seu celular, procure por dispositivos Bluetooth. Conecte-se ao "HC-05" usando a senha padrão `1234` ou `0000`. O LED do módulo passará a piscar lentamente.
  3. Controle pelo App: Abra o "Arduino Bluetooth Control", conecte-se ao HC-05. O LED do módulo ficará aceso direto.
    • Selecioneo gamepad
    • pressione 'quadrado' para LIGAR o LED.
    • pressione 'triângulo' para DESLIGAR o LED.
